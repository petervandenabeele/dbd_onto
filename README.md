# DbdOnto

Ontologies for Dbd.

## Installation

Add this line to your application's Gemfile:

    gem 'dbd_onto'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dbd_onto

## Usage

Used by Dbd and applications that use Dbd.
